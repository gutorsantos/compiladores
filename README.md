# Compiladores

## Para executar:

```bash
./build.sh
./compiler example/fibonacci.simple
fpgrars --no-video output.s
```

### Para limpar:

```bash
rm -r generated
```

### Os arquivos gerados podem ser verificados em:

```bash
./compiler examples/1.simple
```

### Tradução para RISC V

```
RV32IMF
```
